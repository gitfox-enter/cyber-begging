# spare-change

一封信 · 一份支持。

> "Spare some change?"

一封匿名信，讲述一个普通学生勉力支撑的生活。如果您觉得有共鸣，可以随意支持。

## 使用

页面纯静态，图片文件替换即可：

```
spare-change/
├── index.html          # 页面主体
├── wechat.jpg          # 微信收款码
├── alipay.jpg          # 支付宝收款码
├── alipay-gift.jpg     # 支付宝红包码
└── begging.gif         # 动图装饰
```
